6. grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));

display: grid;
/* minmax()函数产生一个长度范围，表示长度就在这个范围之中。它接受两个参数，分别为最小值和最大值。 minmax(100px, 1fr)表示列宽不小于100px，不大于1fr。 分配逻辑：
1. 按 fr 比例初步计算宽度
2. 检查 minmax 的最小值约束
3. 如果不满足，先满足最小值
4. 剩余空间再按fr 比例分配 */
grid-template-columns: 1fr 1fr minmax(100px, 1fr);
